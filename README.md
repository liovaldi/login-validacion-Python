Sistema de validación de usuario
Aplicacion desarrollada en Python que simula un sistema de validacion de usuario y contraseña mediante interaccion en consola.

Funcionalidades:
a)Validacion de usuario y contraseña 
b)Control de acceso
c)Mensajes de error o exito
d)Modificacion de constraseña

Logica implementada:
a)Uso de variables para almacenamiento
b)Validacion de datos ingresados
c)Estructuras condicionales
d)Control de intentos

Tecnologias:
Python - Visual Studio Code

Objetivo:
Aplicar la logica de programacion en un sistema basico de autenticacion.
